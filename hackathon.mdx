whistle is a submission to the OKX **X Cup** — a World Cup-themed hackathon
on X Layer, judged on a **dual track**: AI judges and human judges score
independently across on-chain data, code quality, innovation and market
potential, then the scores are combined.

This page is written so both can verify the project quickly.

## The one-line pitch

The World Cup is too big to watch alone, so you fund three AI agents who
work it for you onchain on X Layer — scouting moments, pricing markets, and
managing a nation.

## How we score against the criteria

| Criterion | Where whistle delivers |
|---|---|
| **Innovation** | "Fund an agent, don't click buttons." LLM-generated matches and agent reads, not scripted heuristics — a differentiated angle on World Cup engagement. |
| **Market potential** | A clean path from World Cup traffic to X Layer transactions: every funding and agent action is an onchain tx in OKB. |
| **Completion** | Live deploy (frontend + API + 5 contracts), real LLM output in production, onchain-verifiable funding, full test + lint + build green. |
| **Demo (bonus)** | A 1–3 min walkthrough of the funding → read → simulate arc. |
